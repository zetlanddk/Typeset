How to get it

npm install typeset

The code is also avail­able on Github.

How to use it

var typeset = require('typeset');
var html = '<p>"Hello," said the fox.</p>';
var output = typeset(html);

Modify type­set.css (<1kb) to match the met­rics of your font and in­clude it on your page. Learn more about the op­tions on Github.
